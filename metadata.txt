about=Plugin QGIS de contrôle qualité, comparaison et mise à jour des données de poteaux aériens dans le cadre des projets FTTH. Supporte NGE (BDD PostgreSQL) et Axione (GraceTHD).

    7 modules d'analyse :
    - MAJ BD : Import Excel FT-BT KO, MAJ SQL directe PostgreSQL
    - Vérification CAP_FT : Poteaux FT QGIS vs Fiches Appuis
    - GESPOT vs C6 : Comparaison CSV GESPOT vs Excel C6, 11 critères
    - Vérification COMAC : Poteaux BT, câbles, boîtiers, portées PCM, schémas techniques
    - C6 vs BD : Annexes C6 vs base QGIS
    - Police C6 : Analyse détaillée C6 vs BDD (câbles, boîtiers, attaches)
    - C6-C3A-C7 vs BD : Croisement annexes contractuelles

    Rapport Excel unifié avec tableau de bord, glossaire et dictionnaire de données ISO/IEC 11179.

    Tables requises : infra_pt_pot, etude_cap_ft, etude_comac. GraceTHD pour Axione.

tracker=https://github.com/asoumare/PoleAerien/issues
repository=https://github.com/asoumare/PoleAerien
